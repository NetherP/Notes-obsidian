# Hacker methodology
**The Process that pentesters follow is summarized in the following steps:**

1.  **Reconnaissance**
2.  **Enumeration/Scanning**
3.  **Gaining Access**
4.  **Privilege Escalation**
5.  **Covering Tracks**
6.  **Reporting**
